Mrroot501 CLI Usage
===

```bash
$ export PATH=$HOME/Projects/Golang-Core/cli/bin:$PATH
$ mrroot501-cli --help 
mrroot501 is a cli tool.

Usage:
  mrroot501 [flags]
  mrroot501 [command]

Available Commands:
  help        Help about any command
  me          Print the information about me.
  timenow     Print timenow.
  weather     Print weather information.

Flags:
  -h, --help   help for mrroot501

Use "mrroot501 [command] --help" for more information about a command.
```
